# tenses

### present continuous

- `S + am/are/is + V-ing`
- use with time periods around now (today, this week, this year, ...)
- talk about a change that has started to happen
	- getting, becoming
	- changing, improving
	- starting, beginning
	- increasing, rising, falling, growing
- `always + V-ing` -> negative meaning
- `am/are/is + being` -> behavior now

### present simple

- `S + V-inf`
- use these words with simple form
	- apologize, advise, insist, agree, refuse, suggest
	- like, want, need, prefer
	- know, realize, understand, recognize
	- believe, suppose, remember, mean
	- belong, fit, contain, consist, seem
	- think (thinking = consider)
	- see, hear, smell, taste, look, feel

### present perfect = how many/how much? -> completed action
- `S + have/has + V-pp`
- use with: just (short time ago), already (before expected), yet (until now - negative/question)
- have/has been to = visited
- not use **finished time** (past): yesterday, last year, ...

### present perfect continuous = how long? -> happening action
- `S + have/has been + V-ing`
- use for/since to say how long something has been happening

### past simple
- `S + V-ed`
- chronological order

### past continuous
- `S + was/were + V-ing`
- something happened (past simple) in the middle of something else (past continuous)

### past perfect
- `S + had + V-pp`
- non-linear order

### past perfect continuous
- `S + had been + V-ing`
